# ボーダーをデザインする

## 学ぶ理由

なぜ学ぶ必要があるのか。この課題が解決するべき問題または達成するべき目標は何か。

## 知識課題

### 課題１

以下のCSSプロパティについて調査し、各プロパティの役割を説明してください。

- `background-color`
- `background-image`
- `background-repeat`
  - `repeat-x`
  - `repeat`
  - `no-repeat`
- `background-position`
- `background-attachment`
- `background`

## 実践課題

上記で学習したプロパティを全て使用したHTMLとCSSファイルを作成してください。
