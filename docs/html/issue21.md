# 入力タイプを学ぼう

## 学ぶ理由

なぜ学ぶ必要があるのか。この課題が解決するべき問題または達成するべき目標は何か。

## 知識課題

- 次の`<input>`要素の `type` 属性について、それぞれの目的と使用シーンを具体的な例を挙げて説明してください。
  - `button`
  - `checkbox`
  - `color`
  - `date`
  - `datetime-local`
  - `email`
  - `file`
  - `hidden`
  - `image`
  - `month`
  - `number`
  - `password`
  - `radio`
  - `range`
  - `reset`
  - `search`
  - `submit`
  - `tel`
  - `text`
  - `time`
  - `url`
  - `week`

## 実践課題

- 上記で挙げたすべての`type`属性を使用し、各`type`がどのように表示され、動作するかを確認できる`HTML`コードを作成してください。
